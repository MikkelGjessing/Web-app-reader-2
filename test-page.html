Web App Reader Extension - Test Page

Purpose: This page is for testing the Web App Reader Chrome Extension after loading it into Chrome.

Step 1: Load the Extension

  1. Open Chrome and navigate to chrome://extensions/
  2. Enable "Developer mode" (toggle in top-right corner)
  3. Click "Load unpacked"
  4. Select the extension directory (the folder containing manifest.json)
  5. The extension icon should appear in your toolbar

Step 2: Configure the Extension

  1. Click the extension icon - you should see a placeholder message
  2. Click "Open Options" in the overlay OR right-click the extension icon and select "Options"
  3. Enter a web app URL (try one of these):
    • https://keep.google.com
    • https://www.notion.so
    • https://trello.com
    • https://chat.openai.com
  4. Optionally check "Show overlay by default"
  5. Click "Save Settings"

Step 3: Test the Overlay

  1. Reload this page (or visit any other website)
  2. Click the extension icon in the toolbar
  3. The overlay should slide in from the right side
  4. The configured web app should load in the iframe
  5. Click the × button to close the overlay
  6. Click the extension icon again to reopen it

What to Verify:

  • ✓ Overlay appears on the right side of the page
  • ✓ Overlay is exactly 16.666% (1/6) of the viewport width
  • ✓ Overlay slides in/out smoothly
  • ✓ Iframe loads the configured web app
  • ✓ Header shows "Web App Reader" title
  • ✓ Close button (×) works correctly
  • ✓ Extension icon toggles the overlay
  • ✓ Options page saves and loads settings
  • ✓ Overlay doesn't interfere with page content
Note: If you encounter any issues, check the browser console for errors:
  • Page Console: Press F12 or Ctrl+Shift+I
  • Extension Service Worker: Go to chrome://extensions/ → Find "Web App Reader" → Click "service worker"
